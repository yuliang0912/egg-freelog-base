# freelog-base

    飞致网络基于eggjs开发微服务的基础框架.

## QuickStart

```bash
$ npm install egg-freelog-base
```

```js
// {app_root}/config/plugin.js
module.exports = {
    eggfreelogBase: {
        enable: true,
        package: "egg-freelog-base",
    }
}
// package.json
{
    "framework": "egg-freelog-base"
}
```
